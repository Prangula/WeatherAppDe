A modern Android weather app built with **Kotlin**, **Jetpack Compose**, **MVI** and **Clean Architecture**.  
It fetches real-time weather data based on your current location.

## Features

- Fetches hourly weather data using **Open-Meteo API**  
- Displays **current weather** and **hourly forecast**  
- Shows **loading** and **error states**  
- **Clean Architecture** with MVI + state management  
- Built with **Jetpack Compose** for a modern UI  
- Fully **responsive and themable**

## Tech Stack

- **Kotlin**  
- **Jetpack Compose**  
- **Retrofit + Gson** for API calls  
- **Koin** for dependency injection  
- **Coroutines + StateFlow** for asynchronous state management  
- **Location Services** for fetching device location
